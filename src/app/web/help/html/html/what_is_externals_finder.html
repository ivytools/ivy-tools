Subversion Externals Finder

What exactly is the Subversion Externals Finder?

This is a quick little tool which looks at your Subversion repository(ies) and tells you which repositories have externals on other repositories, and where. It can do a shallow (just going down to trunk, branches, and tags), or a deep (recursive) search, and produces a graphical report.

See the "using" section of this help document for more documentation on how to use the app.
