Ivy Grapher

What exactly is the Ivy Grapher?

Ivy Grapher is a program I've written to make visualizing how your Ivy config file brings in dependencies. Specifically, to determine if the Ivy file is explicitly calling dependencies which are already brought in by another dependency, or perhaps calling an older revision which is evicted by another dependency.

You point IvyGrapher at the Ivy report generated when you run Ivy - it's in an .ivy directory, typically in your home directory. It takes that report and generates an image showing the replationships between the dependencies.
