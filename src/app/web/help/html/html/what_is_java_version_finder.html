Java Version Finder

What exactly is the Java Version Finder?

Ever get those "class file has wrong version 49.0, should be 48.0" type errors? Wonder which jar of the dozens in your classpath is the offender?

This is a quick little tool which looks at your classpath and tells you which jars have which versions. It will examine the given directories recursively and produce a report of all jars and classes found, along the Java versions. You can highlight anything above a threshold.

See the "using" section of this help document for more documentation on how to use the app.
