Ivy Builder

What exactly is the Ivy Builder?

Ivy Builder is a program I've written to make adding libraries to Ivy easy - as it should be.

Ivy is a fantastic tool for library mangement, but repository management is a pain. Hand-crafting XML, running maint, I manage to have a typo somewhere almost every time. So, I decided to write a program to let me select the files I want to add, and graphically select the dependencies already in my repository - no chance to screw it up!

When you're done, all you have to do is commit (we use subversion. If you just use HTML, you don't even have to commit!).

See the "using" section of this help document for more documenation on how to use the app.
