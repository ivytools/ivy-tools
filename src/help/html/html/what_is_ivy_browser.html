Ivy Browser

What exactly is the Ivy Browser?

Ivy Browser is a program I've written to make searching an Ivy repository easy - as it should be.

Ivy is a fantastic tool for library mangement, but finding what's in the repository can be a pain, and I hate hand-editing the ivy.xml config files. I wanted a tool that would tell me what's in the repository, and let me search for things by org, module, or revision.

See the "using" section of this help document for more documenation on how to use the app.
