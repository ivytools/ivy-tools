Ivy Formatter

Using Ivy Formatter

When you start the program, you get a big, empty textbox:

If you haven't done so already, copy the contents of the Ivy file you want to format into your copy/paste buffer.

Now, you can just click the "Format Ivy Text" button - you don't even need to paste into the text box, it'll just take whatever's in the buffer (although you can paste into the box if you wish).

For illustration purposes, we'll show the app as if someone pasted the text into the box:

However you started, clicking the "Format Ivy Text" button got the text, formatted it, and pasted it back into your copy/paste buffer:
